### How to Run the Employee Payroll System

1. **Clone the Repository**: First, clone the repository to your local machine using the following command:
   ```
   git clone https://github.com/anil-k-rajamoni18/cs-course.git
   ```

2. **Navigate to the Project Directory**: Change your current directory to the project folder:
   ```
   cd cs-course/c-lang/projects/employee_payroll
   ```

3. **Compile the Code**: Use a C compiler like `gcc` to compile the source code. Run the following command:
   ```
   gcc -o employee_payroll main.c employee.c
   ```

4. **Run the Executable**: After successful compilation, run the executable to start the Employee Payroll System:
   ```
   ./employee_payroll
   ```

5. **Follow On-Screen Instructions**: The program will display a menu with various options. Follow the on-screen instructions to perform different employee payroll operations such as adding an employee, viewing employee details, and generating payslips.
