### How to Run the Bank Management System

1. **Clone the Repository**: First, clone the repository to your local machine using the following command:
   ```
   git clone https://github.com/anil-k-rajamoni18/cs-course.git
   ```

2. **Navigate to the Project Directory**: Change your current directory to the project folder:
   ```
    cd cs-course/c-lang/projects/bank_mgmt
    ```

3. **Compile the Code**: Use a C compiler like `gcc` to compile the source code. Run the following command:
    ```
    gcc main.c account.c transaction.c -o bank_mgmt
    ``` 

4. **Run the Executable**: After successful compilation, run the executable to start the Bank Management System:
    ```
    ./bank_mgmt
    ```

5. **Follow On-Screen Instructions**: The program will display a menu with various options. Follow the on-screen instructions to perform different banking operations such as creating an account, depositing money, withdrawing money, and viewing account details.
