### âœ³ï¸ Reverse a String (Manual)
```c

#include <string.h>

    char str[50], rev[50];
    int i, len;

    printf("Enter string: ");
    gets(str);

    len = strlen(str);

    rev[i] = '\0';

    printf("Reversed: %s", rev);
    return 0;
}
```

## ğŸ”¹ 14. Important Points

- âœ… String constants are stored in read-only memory.
- âœ… Character arrays can be modified; string literals cannot.
- âœ… Always ensure space for `'\0'` in arrays.
- âœ… Use `fgets()` instead of `gets()` for safety.
- âœ… `strcmp()` returns:
  - `0` â†’ Equal
  - `<0` â†’ s1 < s2
